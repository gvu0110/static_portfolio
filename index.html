First of all, I love writing code and software development is my thing. Since compiling my very first C program to a micro-controller to control the LED, I have been pursuing the idea of using computer software to automate things and make our life easier. Having strong background in computer networking with CCIE #61090 Certification, AWS Solutions Architect Certification Having experience in configuring networking devices (Routers, Switches, Firewalls, Wireless AP especially Cisco/Juniper devices), operating many services (Active Directory, Website/Database Server, Load Balancing, VPN, etc), and working with multiple OS (Linux - Ubuntu, Red Hat, CentOS, and MacOS (JAMF Pro)) and virtualization technology (VMware, Hyper-V). Pursuing career as a Software Engineer, I studied data structures and algorithms. I also acquire deep understanding and hands-on practice in containerization technology (Docker, Kubernetes), public cloud platform (AWS - EC2, ELB, RDS, S3, VPC, Route53, CloudFront, etc and Microsoft Azure), RESTful API, source code management (Git, GitHub), continuous delivery pipelines (Jenkins), programming language (Java SE (Swing), Python (Flask, Paramiko), Golang (Echo), C/C++), scripting and automation tools (Terraform, Ansible, Bash, Shell Script) I were certified with other vendor certifications (Cisco CCNA+CCNP, Juniper JNCIA and Microsoft MCSA) since I was in my college.

Even during my free time, I'm continuously self-taught with computer sciences online courses. Learning new technology and programming languages keep me up at night in order to acquire knowledge every day
